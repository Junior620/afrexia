# Sanity Sample Data

This directory contains sample data for populating the Afrexia CMS with initial content.

## Contents

- `products.json` - Sample product data for all commodities
- `certifications.json` - Sample certification data
- `team-members.json` - Sample team member data
- `blog-posts.json` - Sample blog post data
- `blog-categories.json` - Sample blog category data
- `resources.json` - Sample resource data

## Usage

To populate Sanity with this sample data:

1. Start the Sanity Studio: `npm run dev` and navigate to `/studio`
2. Use the Sanity Studio interface to manually create documents based on these samples
3. Alternatively, use the Sanity CLI or API to import data programmatically

## Notes

- All sample data includes both French and English translations
- Images are referenced as placeholders - replace with actual images
- Adjust data as needed for your specific requirements
- Ensure all required fields are populated before publishing
