# Managing Blog Posts

This guide explains how to create and manage blog posts in the Afrexia CMS.

## Overview

Blog posts help establish Afrexia as a thought leader in agricultural commodities. They provide valuable content for buyers, improve SEO, and drive organic traffic to the website.

## Blog Post Structure

A blog post contains:

- **Basic Information**: Title, slug, author
- **Content**: Featured image, excerpt, main content
- **Organization**: Categories and tags
- **Publishing**: Date and reading time
- **SEO**: Meta tags for search engines
- **Workflow**: Draft/Review/Published status

## Creating a New Blog Post

### Step 1: Access Blog Posts

1. Click **"Blog Post"** in the sidebar
2. Click **"+ Create"** button

### Step 2: Basic Information

#### Title (Required)

Enter the article title in both languages:
- **French**: e.g., "Tendances du Marché du Cacao en 2024"
- **English**: e.g., "Cocoa Market Trends in 2024"

**Tips**:
- Make it compelling and specific
- Include keywords for SEO
- Keep it under 60 characters
- Use title case in English

#### Slug (Required)

Generate URL-friendly slugs:
1. Click **"Generate"** for French slug
2. Click **"Generate"** for English slug

**Example**:
- French: `tendances-marche-cacao-2024`
- English: `cocoa-market-trends-2024`

#### Translation ID (Required)

Enter a unique identifier:
- Format: `blog-topic-year`
- Example: `blog-cocoa-market-2024`

**Tips**:
- Use descriptive IDs
- Include year for time-sensitive content
- Keep it consistent with your naming convention

#### Author (Required)

Select the article author:
1. Click **"Select"**
2. Search for team member name
3. Click to select

**Tips**:
- Author must be created first in Team Members
- Choose the actual writer or subject matter expert
- Author photo and bio will appear on the article

### Step 3: Featured Image (Required)

Upload the main article image:

1. Click **"Upload"** or drag and drop
2. Crop/adjust if needed
3. **Alt Text** (Required): Describe the image
   - Example: "Cocoa beans drying in the sun"

**Image Guidelines**:
- Format: JPG or PNG
- Size: 1200x630px (optimal for social sharing)
- File size: Under 500KB
- High quality, relevant to article topic

**Tips**:
- Use professional, high-quality images
- Ensure image relates to article content
- Consider social media preview
- Avoid text-heavy images

### Step 4: Excerpt (Required)

Write a brief summary (150-200 characters):

**French**:
```
Analyse des principales tendances qui façonnent le marché 
mondial du cacao en 2024, incluant la demande croissante 
pour le chocolat premium.
```

**English**:
```
Analysis of the main trends shaping the global cocoa market 
in 2024, including growing demand for premium chocolate.
```

**Tips**:
- Summarize the main point
- Make it enticing to read more
- Include key topics covered
- Keep it concise but informative

### Step 5: Main Content (Required)

Write the article body using the rich text editor:

#### Formatting Options

- **Headings**: Use H2 for main sections, H3 for subsections
- **Bold**: Emphasize important points
- **Italic**: For emphasis or foreign terms
- **Lists**: Bulleted or numbered lists
- **Links**: Add external or internal links
- **Quotes**: Use blockquotes for citations

#### Content Structure

**Introduction** (1-2 paragraphs):
- Hook the reader
- State the main topic
- Preview what's covered

**Body** (3-5 sections with H2 headings):
- Break content into logical sections
- Use subheadings (H3) for subsections
- Include data, examples, and insights
- Add lists for easy scanning

**Conclusion** (1-2 paragraphs):
- Summarize key points
- Provide actionable takeaways
- Include call-to-action if appropriate

#### Writing Tips

**Do**:
- Write in a professional but accessible tone
- Use industry terminology correctly
- Provide valuable, actionable information
- Include data and examples
- Break up long paragraphs
- Use active voice

**Don't**:
- Use overly promotional language
- Make unsubstantiated claims
- Copy content from other sources
- Use jargon without explanation
- Write walls of text

#### Example Structure

```markdown
# Introduction
Brief overview of the topic and why it matters.

## Main Section 1: Market Overview
Current state of the market with data and trends.

### Subsection: Regional Differences
Specific details about different regions.

## Main Section 2: Key Trends
- Trend 1: Description and impact
- Trend 2: Description and impact
- Trend 3: Description and impact

## Main Section 3: Implications for Buyers
What this means for B2B buyers and how to respond.

## Conclusion
Summary of key points and next steps.
```

### Step 6: Categories (Required)

Select one or more categories:

1. Click **"Select"**
2. Choose relevant categories
3. Add multiple if applicable

**Available Categories**:
- **Market News**: Market trends and analysis
- **Sustainability**: Environmental and social topics
- **Quality and Certifications**: Standards and compliance
- **Practical Guides**: How-to and educational content
- **Traceability**: Supply chain and EUDR topics

**Tips**:
- Choose 1-2 primary categories
- Don't over-categorize
- Categories help readers find related content

### Step 7: Tags (Optional)

Add relevant tags:

1. Type tag name
2. Press Enter
3. Repeat for more tags

**Example tags**:
- Product names: "cacao", "café", "poivre"
- Topics: "marché", "durabilité", "qualité"
- Regions: "Cameroun", "Afrique"
- Concepts: "EUDR", "certifications", "export"

**Tips**:
- Use 3-7 tags per article
- Use lowercase
- Be specific but not too narrow
- Tags improve search and related articles

### Step 8: Publishing Details

#### Published At (Required)

Set the publication date and time:
1. Click the date field
2. Select date from calendar
3. Set time (optional)

**Tips**:
- Use current date/time for immediate publication
- Schedule future posts by setting future date
- Maintain consistent publishing schedule

#### Reading Time (Optional)

Enter estimated reading time in minutes:
- Example: 5, 8, 12

**Tips**:
- Count ~200 words per minute
- Round to nearest minute
- Helps readers decide if they have time to read

### Step 9: SEO Optimization

#### Meta Title

Create search-friendly titles (50-60 characters):
- **French**: "Tendances du Marché du Cacao 2024 | Afrexia Blog"
- **English**: "Cocoa Market Trends 2024 | Afrexia Blog"

**Tips**:
- Include main keyword
- Add "| Afrexia Blog" at end
- Make it compelling for clicks
- Keep under 60 characters

#### Meta Description

Write search result descriptions (150-160 characters):
- **French**: "Découvrez les principales tendances du marché du cacao en 2024 et leur impact sur l'industrie du chocolat."
- **English**: "Discover the main cocoa market trends in 2024 and their impact on the chocolate industry."

**Tips**:
- Summarize the article value
- Include main keyword
- Add call-to-action
- Keep under 160 characters

### Step 10: Workflow Status

Set the document status:
- **Draft**: Work in progress
- **In Review**: Ready for editorial review
- **Published**: Live on website

### Step 11: Publish

1. Review all content
2. Proofread for typos and errors
3. Check both language versions
4. Click **"Publish"** button
5. Article appears on website within 5 minutes

## Editing Existing Blog Posts

### Finding a Post

1. Click **"Blog Post"** in sidebar
2. Use search bar to find by title
3. Or filter by category/author
4. Click the post to open

### Making Changes

1. Edit any field
2. Changes auto-save
3. Click **"Publish"** to update website

### Updating Old Posts

To keep content fresh:
1. Review posts older than 6 months
2. Update statistics and data
3. Add new information
4. Update publication date if major changes
5. Republish

## Blog Post Checklist

Before publishing, ensure:

- [ ] Title in French and English
- [ ] Slugs generated for both languages

- [ ] Author selected
- [ ] Featured image uploaded with alt text
- [ ] Excerpt written in both languages
- [ ] Main content written in both languages
- [ ] Content is well-structured with headings
- [ ] At least 1 category selected
- [ ] Relevant tags added
- [ ] Publication date set
- [ ] Reading time estimated
- [ ] SEO meta title and description in both languages
- [ ] Content proofread for errors

## Best Practices

### Content Quality

- **Original content**: Write unique, valuable articles
- **Research thoroughly**: Use reliable sources
- **Cite sources**: Link to data and references
- **Update regularly**: Keep information current
- **Proofread carefully**: Check for errors in both languages

### SEO

- **Keyword research**: Use terms buyers search for
- **Natural language**: Don't stuff keywords
- **Internal links**: Link to relevant products and pages
- **External links**: Link to authoritative sources
- **Image optimization**: Use descriptive alt text

### Engagement

- **Compelling headlines**: Make readers want to click
- **Strong opening**: Hook readers immediately
- **Scannable content**: Use headings, lists, short paragraphs
- **Visual breaks**: Break up text with images or quotes
- **Clear takeaways**: Provide actionable insights

### Publishing Schedule

- **Consistency**: Publish regularly (weekly or bi-weekly)
- **Quality over quantity**: Better to publish less often with high quality
- **Plan ahead**: Maintain an editorial calendar
- **Timely topics**: Cover current industry issues

## Common Issues

### Post Not Appearing on Website

**Possible causes**:
- Workflow status not "Published"
- Publication date in the future
- Missing required fields

**Solution**:
- Check workflow status
- Verify publication date is in the past
- Ensure all required fields are complete

### Related Articles Not Showing

**Possible cause**:
- No shared categories or tags with other posts

**Solution**:
- Add relevant categories and tags
- Ensure other posts have similar categories/tags

### Poor Search Ranking

**Possible causes**:
- Weak SEO optimization
- Duplicate content
- Thin content (too short)

**Solution**:
- Optimize meta tags with keywords
- Write original, unique content
- Aim for 800+ words per article
- Add internal and external links

## Analytics and Performance

- Update underperforming posts
- Promote high-performing content
