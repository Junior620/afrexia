## Overview

The Afrexia CMS is built on Sanity, a modern headless CMS that provides:

- **Multilingual Support**: Manage content in French and English
- **Real-time Collaboration**: Multiple editors can work simultaneously
- **Version History**: Track all changes and revert if needed
- **Flexible Content**: Rich text editing with custom components
- **Media Management**: Upload and optimize images and documents
- **Workflow States**: Draft → In Review → Published

## Content Types

The CMS manages the following content types:

| Content Type | Description | Used For |
|-------------|-------------|----------|
| **Products** | Agricultural commodities | Product catalog and detail pages |
| **Certifications** | Quality certifications | Product pages, quality page |
| **Blog Posts** | Articles and news | Blog section |
| **Blog Categories** | Article categories | Blog organization |
| **Team Members** | Staff profiles | About page, blog authors |
| **Resources** | Downloadable documents | Resources page, product pages |
| **Pages** | Static page content | Custom pages |

## User Roles

The CMS supports three user roles:

### Administrator
- Full access to all content
- Can create, edit, publish, and delete
- Manages user permissions
- Configures CMS settings

### Editor
- Can create and edit content
- Can submit content for review
- Cannot publish without approval
- Cannot delete published content

### Viewer
- Read-only access
- Can view all content
- Cannot make changes
- Useful for stakeholders and reviewers
